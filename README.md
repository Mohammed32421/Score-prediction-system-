# Score-prediction-system-
The following project, “Score Prediction System,” was created and developed in 2026 by two students of Beddellee Boarding School,Yonas Amare, Eyob Demelash and Mohammed Ahmed.

The project features a machine learning system designed to predict students’ mathematics scores. It includes advanced data preprocessing, hyperparameter tuning using RandomizedSearchCV, and performance comparison across seven different machine learning models, achieving an accuracy score of R² > 0.88.

The system was further deployed through a Flask-based web application and JSON API, incorporating logging and error-handling mechanisms for improved reliability and performance.
